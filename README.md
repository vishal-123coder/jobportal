# Job Portal Backend System

## Overview
Backend system for Job Portal built using Spring Boot with authentication, role-based access, and job management features.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- PostgreSQL
- JPA/Hibernate
- Maven
- Postman (API Testing)

## Features
- User Registration & Login
- JWT-based Authentication
- Role-based Access (Recruiter / User)
- Job Posting (Recruiter)
- Job Listing (User)
- Secure REST APIs
- Input Validation

## API Testing
All APIs tested using Postman.

## Screenshots
- Login API (JWT Token)
- Job Creation API
- Job Fetch API
- Application Logs (Spring Boot)

## How to Run
1. Clone repository
2. Configure PostgreSQL DB
3. Run Spring Boot application
4. Use Postman for API testing
